# Bài tập Dart - Số nguyên dương lẻ nhỏ hơn 100

In ra màn hình tất cả các số nguyên dương lẻ nhỏ hơn `100`.

```dart
void main() {
  for (int i = 0; i < 100; i++) {
    if (i % 2 == 1) print(i);
  }
}
```
